Html5MvcTemplates by srkirkland (Scott Kirkland)

Html5 ASP.NET MVC Input Templates

Html5 Object Templates for ASP.NET MVC 2 and 3. This project adds the new html5 input types, such as email, tel, and url, to the MVC EditorTemplates. See https://github.com/srkirkland/Html5MvcTemplates for more details.

Dependencies

ASP.NET MVC 3

Install

For now, just copy the Views/Shared/EditorTemplates folder into your project. Soon, there will be a shiny NuGet package which does the same thing.
